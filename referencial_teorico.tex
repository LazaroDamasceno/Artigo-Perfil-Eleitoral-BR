\section{Referencial teórico}

A linguagem de programação Python foi utilizada para a análise exploratória de dados (AED) e para a modelagem de algoritmos de aprendizado de máquina (AM). Para a AED, foram utilizadas as bibliotecas pandas, polars, dask, numpy, matplotlib e seaborn. Para o AM, foi utilizada a biblioteca sklearn.

O uso conjunto de polars e dask com pandas é justificado porque pandas não foi desenvolvido para lidar com volumes medianos e grandes. A biblioteca polars aguenta conjuntos de dados medianos; já o dask, conjuntos de dados grandes.
